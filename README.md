# Como rodar o código

1. Instalar Node.js (se não tiver instalado na máquina) /*link do node*/

2. Clonar repositório na sua máquina
    a) No botão verde chamado 'Code', clique na setinha e depois 'Open with github desktop'

3. Instalar dependências do node
    a) No explorador de arquivos do Windows, procure a pasta onde foi clonado o repositório
    b) Abra o CMD por meio da 'barrinha de pesquisa' (só apagar oq ta escrito e escrever "cmd")
    c) No CMD, digite 'npm install' e 'depois npm run dev' para abrir o site no navegador
